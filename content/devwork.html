What Would Yeezus Do?

WWYD is a simple web application that plays random Kanye West music videos. It first started off as a project to learn about the MEAN stack (MongoDB, ExpressJS, AngularJS, and Node.js), which is a bit excessive for such a simple application, but it was for learning purposes. Although I got the MEAN stack working, I was unable to work with the Youtube API nicely within AngularJS so I decided to scrap it and go with the simpler Javascript/HTML5 approach. However, I did learn quite a bit by using MongoDB and Node.js and how to quickly build a fullstack Javascript application, how to deploy it using 5apps, and got to work a little bit with the Twitter and Facebook APIs. I hope to use this the MEAN stack successfully in a future project.
